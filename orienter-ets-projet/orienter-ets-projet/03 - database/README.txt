Pour utiliser la base de donnée avec MySQL : 

Avoir MySQL de downloaded. 
Ajouter à la variable d'environnement PATH. C:\Program Files\MySQL\MySQL Server 8.0\bin
Vérifiez que le path est le bon.

Naviguer dans le folder 03-database depuis cmd.
Entrer la commande dans le cmd : mysql -u root -p storingorientation
Entrez le mot de passe : root.

Pour exporter la DB lorsque vous avez fait des changements : 
mysqldump -u root -p storingorientation
